# Call Demand Analytics & Weekly Insights

## Summary

Extract and categorize client demands from call transcripts in real-time, generate weekly performance reports with KPIs, and provide AI-powered recommendations personalized to the professional's domain.

## Architecture: Real-time extraction + batch reporting

### Real-time Extraction (Webhook)

In the existing `/api/retell/webhook` handler (event `call_analyzed`), after lead scoring:

1. Send transcript + `CompanyProfile.activity` to Claude Haiku
2. Prompt extracts JSON: `[{ category, label, details, urgency }]`
3. Insert `CallDemand` rows in DB
4. Non-blocking: if LLM call fails, log error and continue

### Weekly Cron Job

**Schedule:** Every Monday at 8:00 AM

**API route:** `/api/cron/weekly-report` secured by token (Vercel Cron)

**Process per org with calls in the previous week:**

1. Aggregate `CallDemand` for the week (Monday-Sunday)
2. Compute KPIs: total calls, total demands, top categories, avg sentiment, avg duration, completion rate
3. Get `CompanyProfile.activity` (or infer from categories if empty)
4. Call Claude Sonnet with full context to generate 3-5 personalized recommendations
5. Save `WeeklyReport` to DB
6. Send HTML email via Resend with: KPIs, top 3 categories, recommendations, CTA to dashboard

### AI Recommendation Engine

**Context sent to LLM:**
- Profession (activity)
- Period
- KPIs: totalCalls, totalDemands, completionRate, avgSentiment, avgDuration
- Top categories with counts and percentages
- Evolution vs previous week (category-level variations)

**Recommendation types:**
- `optimization`: improve an overloaded process
- `opportunity`: exploit a positive trend
- `alert`: flag a problem or risk

**Output:** `[{ title, description, priority, type }]`

**Examples:**
- Accountant, 60% payslip requests â†’ "Optimize payslip delivery â€” 60% of your calls are about this. Consider a self-service portal."
- Doctor, dermatology +40% â†’ "Strong demand in dermatology (+40%) â€” you have notable influence. Consider expanding dedicated time slots."
- Lawyer, 30% negative sentiment â†’ "Satisfaction alert: 30% of callers express dissatisfaction. Review transcripts to identify friction points."

### Technical Choices

- **Extraction LLM:** Claude Haiku (fast, cheap, simple structuring task)
- **Recommendations LLM:** Claude Sonnet (needs deeper reasoning for personalized advice)
- **Email service:** Resend (lightweight, good free tier, native Next.js integration)
- **Profession detection fallback:** If `CompanyProfile.activity` is empty, LLM infers from transcript categories
- **Cron:** Vercel Cron calling secured API route
